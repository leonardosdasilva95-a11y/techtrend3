Como funciona uma Rede Neural?

As redes neurais artificiais são inspiradas no cérebro humano. Cada "neurônio" recebe informações, processa e envia para o próximo. Essa animação mostra como os dados fluem entre camadas de uma rede neural.

Curiosidade
