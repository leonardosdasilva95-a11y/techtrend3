O que é Machine Learning?

Machine Learning (Aprendizado de Máquina) é uma área da Inteligência Artificial que permite que sistemas aprendam com dados e melhorem com a experiência — sem serem programados diretamente para cada tarefa.

Como funciona?

O sistema recebe dados de entrada, identifica padrões e gera previsões ou decisões com base nesses padrões. Quanto mais dados, melhor o aprendizado.

Curiosidade

O termo “Machine Learning” foi usado pela primeira vez em 1959 por Arthur Samuel, que criou um programa que aprendia a jogar damas.
